About This Project

This project is a full-stack web application built with modern, open-source technologies. It provides image compression and related features in the frontend and a NestJS backend with PostgreSQL for persistent storage.

Angular

Frontend framework used for the UI and client-side logic.

NestJS

Backend framework used to build the REST API and handle business logic.

PostgreSQL

Relational database used for storing application data reliably.
